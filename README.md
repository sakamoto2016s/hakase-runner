# 博士ランナー

博士を操作して障害物を避け、できるだけ長く走り続ける **エンドレスラン** のブラウザゲームです。HTML 1 ファイルで完結します。

## 特徴

- **単一 HTML ファイル**（ビルドやパッケージマネージャ不要）
- **PC + スマホ**（キーボードとタッチの両方を想定）
- **ジャンプ**と**スライディング**で障害物を回避
- 障害物に当たるとゲームオーバー。スコアは生存時間に応じて伸びていきます。

## 遊び方

### ルール

- 障害物に衝突すると **ゲームオーバー** です。
- **スコア**は、生存している時間が長いほど高くなります（実装では秒数やフレームに比例した加算が一般的です）。

### PC（キーボード）

| 操作       | キー                         |
| ---------- | ---------------------------- |
| ジャンプ   | `Space` または `W`         |
| スライディング | `↓`（下矢印）または `S` |

### スマホ（タッチ）

| 操作       | 操作                         |
| ---------- | ---------------------------- |
| ジャンプ   | 画面を **タップ**            |
| スライディング | 画面を **下方向にスワイプ**（または画面下部のスライド用 UI があればそちら） |

## 起動方法

1. リポジトリをクローンまたはダウンロードします。
2. ブラウザで **`index.html`** を開きます（ダブルクリック、またはファイルをブラウザにドラッグ＆ドロップ）。

ローカルサーバは不要です。効果音や外部フォントを読み込む実装にした場合、ブラウザや端末によっては制約が出ることがあります。

## 技術スタック

- HTML / CSS / JavaScript
- 描画は **Canvas 2D** を想定
- 外部ライブラリ・ビルドツールは使用しない方針

## フォルダ構成（想定）

```
hakase-runner/
├── README.md
└── index.html    # ゲーム本体（単一ファイル）
```

画像・音声を Data URL 等で `index.html` に埋め込む場合も、この構成のままで問題ありません。
